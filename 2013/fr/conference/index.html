LA CONFÉRENCE 2013

 

Après le succès des précédentes éditions, Softshake revient pour la quatrième année : Softshake 2013 se déroulera les jeudi 24 et vendredi 25 octobre, à Genève dans les locaux de l'hepia.

Nous attendons plus de 300 participants, francophones et anglophones.

Cette conférence est supportée par l'association des communautés locales :

  • Le Groupe des Développeurs iPhone de Suisse romande
  • Le Geneva JUG
  • Agile Genève
  • La Ruby Brigade de Genève
Pour chacun de ces thèmes, ou tracks, nous dédions une salle, afin de garantir un parfait équilibre, une bonne représentation et d'éviter qu'un thème ne prédomine. Pour cette nouvelle édition, nous avons identifié les thèmes suivants :
  • Agilité
  • Mobilité
  • Java
  • Microsoft
  • Ruby

Nous avons également prévu un track Incubateur afin de pouvoir laisser la parole à des intervenants sur tout autre sujet informatique.
